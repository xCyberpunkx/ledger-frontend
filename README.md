# Ledger — Frontend

Next.js (App Router) + TypeScript + Tailwind + Shadcn UI.

## Quickstart
```bash
npm install
cp .env.example .env.local
npm run dev
```
Runs at `http://localhost:3000`. Expects the backend at
`NEXT_PUBLIC_API_URL` (default `http://localhost:4000/api/v1`).

## Status
V0 — foundation only (placeholder home page). Auth pages land in V1.
